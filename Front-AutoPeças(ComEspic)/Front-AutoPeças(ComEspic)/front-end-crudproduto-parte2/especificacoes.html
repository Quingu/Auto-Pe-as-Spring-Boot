Especificações da API

1. Retornar Dados de um Produto pelo ID

End-Point

URL: http://localhost:8080/produto/list/{id}

Verbo HTTP: GET

Descrição

Este endpoint retorna os dados de um produto específico armazenado na base de dados, utilizando o ID fornecido na URL.

Casos de Uso

Produto encontrado:

Código de status HTTP: 200 OK

{ "id": 1, "nome": "Produto A", "descricao": "Descrição do produto A", "categoria": "Categoria 1", "fabricante": "Fabricante A", "modelo": "Modelo X", "data_fabricacao": "2023-11-10", "preco": 199.99, "quantidade": 20, "lote": "L12345" }

Produto não encontrado:

Código de status HTTP: 404 Not Found

null

2. Adicionar Produto

End-Point

URL: http://localhost:8080/produto/add

Verbo HTTP: POST

Descrição

Este endpoint insere um novo produto na base de dados a partir de um objeto enviado no corpo da requisição.

Casos de Uso

Produto adicionado com sucesso:

Código de status HTTP: 201 Created

{ "id": 101, "nome": "Novo Produto", "descricao": "Descrição do novo produto", "categoria": "Categoria Nova", "fabricante": "Fabricante Novo", "modelo": "Modelo Y", "data_fabricacao": "2024-01-01", "preco": 299.99, "quantidade": 50, "lote": "L98765" }

3. Deletar Produto

End-Point

URL: http://localhost:8080/produto/delete/{id}

Verbo HTTP: DELETE

Descrição

Este endpoint remove um produto da base de dados com base no ID fornecido na URL.

Casos de Uso

Produto deletado com sucesso:

Código de status HTTP: 204 No Content

Produto não encontrado:

Código de status HTTP: 404 Not Found

null

4. Atualizar Produto

End-Point

URL: http://localhost:8080/produto/update/{id}

Verbo HTTP: PUT

Descrição

Este endpoint atualiza os dados de um produto existente na base de dados utilizando o ID fornecido na URL e os novos dados enviados no corpo da requisição.

Casos de Uso

Produto atualizado com sucesso:

Código de status HTTP: 200 OK

{ "id": 1, "nome": "Produto Atualizado", "descricao": "Descrição atualizada do produto", "categoria": "Categoria Atualizada", "fabricante": "Fabricante Atualizado", "modelo": "Modelo Z", "data_fabricacao": "2023-12-01", "preco": 299.99, "quantidade": 40, "lote": "L54321" }
